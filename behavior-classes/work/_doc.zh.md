# 工作

> 职业相关的任务和活动。

## 规则

1. **明确目标**：每个工作任务必须有明确的目标和交付物
2. **时间估算**：预估任务时间，设置截止时间
3. **专注执行**：工作时段避免干扰，保持专注
4. **及时沟通**：遇到问题及时沟通，不要独自死磕
5. **复盘总结**：项目结束后进行复盘

## 经验

- 上午是深度工作黄金时间，处理重要任务
- 把大任务拆分成小任务，降低启动阻力
- 使用番茄工作法保持专注
- 每天列出 3 件最重要的事，优先完成
- 邮件和消息集中处理，避免频繁打断

## 方法

### 番茄工作法

25 分钟专注 + 5 分钟休息的循环。

1. 选择一个任务
2. 设定 25 分钟计时器
3. 专注工作，直到计时器响起
4. 休息 5 分钟
5. 每 4 个番茄后，休息 15-30 分钟

### GTD 工作法

Getting Things Done，把事情做完。

1. 收集：把所有任务收集到收件箱
2. 处理：清空收件箱，任务分类
3. 组织：按项目和情境组织任务
4. 回顾：定期回顾任务列表
5. 执行：按情境和优先级执行任务

# Work

> Career-related tasks and activities.

## Rules

1. **Clear Objectives**: Every work task must have clear objectives and deliverables
2. **Time Estimation**: Estimate task time and set deadlines
3. **Focused Execution**: Avoid distractions during work hours, maintain focus
4. **Timely Communication**: Communicate promptly when encountering problems, don't struggle alone
5. **Review and Summarize**: Conduct post-project reviews

## Experiences

- Morning is the golden time for deep work, handle important tasks then
- Break large tasks into smaller ones to reduce resistance to starting
- Use Pomodoro Technique to maintain focus
- List 3 most important things each day and prioritize them
- Handle emails and messages in batches to avoid frequent interruptions

## Methods

### Pomodoro Technique

25 minutes focus + 5 minutes rest cycles.

1. Select a task
2. Set a 25-minute timer
3. Focus on work until timer rings
4. Rest for 5 minutes
5. After every 4 pomodoros, rest for 15-30 minutes

### GTD Method

Getting Things Done.

1. **Collect**: Gather all tasks into inbox
2. **Process**: Empty inbox, categorize tasks
3. **Organize**: Organize tasks by project and context
4. **Review**: Regularly review task lists
5. **Execute**: Execute tasks by context and priority

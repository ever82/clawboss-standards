# 学习

> 获取新知识和技能的过程，是人类进步的核心驱动力。

## 规则

执行学习类行为时应遵循的规范：

1. **目标明确**：每次学习前必须明确学习目标
2. **时间限制**：单次学习时长不超过 90 分钟，必须休息 10-15 分钟
3. **及时总结**：完成学习后必须做总结笔记，记录收获和疑问
4. **定期复习**：遵循艾宾浩斯遗忘曲线，在学习后 1天/3天/7天/30天 进行复习
5. **输出导向**：学完后必须有输出（笔记、实践、教别人）

## 经验

过往积累的关于学习的经验和洞察：

- 番茄工作法（25+5）对深度学习很有效
- 早上 6-9 点是记忆黄金时间，适合背诵类学习
- 教别人是最好的学习方式（费曼技巧）
- 2024-03: 发现结合实践比纯阅读效率高 3 倍
- 睡前 30 分钟复习，记忆效果提升 40%
- 多维度学习（读+听+写+说）比单一方式效果好 5 倍

## 方法

### 费曼学习法

用最简单的话解释复杂概念，发现理解漏洞。

1. 选择一个概念
2. 想象要教给一个完全不懂的小白
3. 用简单语言讲解，发现卡壳的地方
4. 回去针对性学习卡壳的地方
5. 简化语言，使用类比和图示

### SQ3R 阅读法

适用于书籍和文献阅读。

1. **Survey（浏览）**：快速浏览目录、标题、总结
2. **Question（提问）**：提出你想从中学到什么
3. **Read（阅读）**：带着问题仔细阅读
4. **Recite（复述）**：用自己的话复述要点
5. **Review（复习）**：回顾并整理笔记

### 刻意练习法

针对薄弱环节进行有目的的练习。

1. 明确具体薄弱环节
2. 设计针对性练习任务
3. 获取即时反馈
4. 重复直到形成肌肉记忆

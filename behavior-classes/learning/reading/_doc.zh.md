# 阅读

> 通过阅读书籍、文章、文档等获取知识。

## 规则

1. **预读筛选**：先读目录和总结，判断价值再深入
2. **主动阅读**：带着问题和目的去阅读
3. **标记批注**：重要内容做标记，写下思考和疑问
4. **控制速度**：根据内容难度调整阅读速度
5. **输出笔记**：读完后必须输出结构化笔记

## 经验

- 技术类书籍：先通读一遍建立框架，再精读关键章节
- 非虚构类：重点关注论点、论据和结论
- 小说类：享受阅读过程，不必强求速度
- 使用 Zotero 或 Notion 管理阅读清单
- 同时读 3-5 本书，根据心情切换，保持阅读兴趣

## 方法

### 渐进式阅读

适用于长篇文章和技术文档。

1. **第一遍**：快速浏览，抓大意（5-10分钟）
2. **第二遍**：细读重点，理解逻辑
3. **第三遍**：深入研究，查阅资料

### 主题阅读法

快速掌握一个新领域。

1. 选定主题，收集 5-10 本相关书籍/文章
2. 每本快速浏览，提取核心观点
3. 对比不同作者的观点，形成自己的理解
4. 输出一篇综述文章
